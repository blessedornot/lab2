# Hotel Management System

Система управления бронированием отеля с расчетом стоимости проживания.

## Описание
Позволяет:
- Создавать номера разных типов
- Регистрировать гостей
- Создавать бронирования
- Рассчитывать стоимость проживания
- Экспортировать отчет в XLSX
